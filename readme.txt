Скрипт для анализа github репозитория (GitHub REST API)

Тестировался в Python 2.7,Python 3.6.1

Использована одна сторонняя библиотека texttable (отрисовка консольных таблиц)
установить кё можно из Директории texttable-0.8.2 при помощи сл. команды:

python setup.py install

1)github.py - основной модуль
2)config.py - класс с настройками
3)readme.txt - Информация
